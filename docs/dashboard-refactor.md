# GTD Dashboard Refactor Notes

This is historical context for the dashboard refactor that led to the current dashboard. It is not a source of truth for present-day behavior.

Current dashboard behavior is documented in [spec/03-runtime-behavior.md](../spec/03-runtime-behavior.md) and the dashboard components under `src/components/dashboard/`.

The original plan captured a transition from the older 4-tab dashboard to a broader 5-tab design with more filtering and analytics. Some ideas shipped, some did not, and several of the aspirational items in the old proposal were never implemented.

Use this file only as background when reading old discussions or commit history.
