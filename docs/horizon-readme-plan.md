# Horizon README Modernization Plan

This is historical planning context for the horizon sidebar/README work. The shipped behavior is now documented in [spec/gtd-spec.md](../spec/gtd-spec.md), [spec/02-markdown-schema.md](../spec/02-markdown-schema.md), and [horizon-readme-template.md](./horizon-readme-template.md).

Use this file only as background when tracing the original design discussion.
