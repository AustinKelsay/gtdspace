# GTD Space {{tag}}

## Highlights

- Fixed the macOS Intel release pipeline by excluding the standalone MCP server binary from the signed desktop app bundle.
- Added a release guard that fails macOS packaging if the MCP binary ever appears inside the app bundle again.

## Notable Changes

- The standalone `gtdspace-mcp` launcher now lives in its own Cargo package, so the desktop release builds only package the GTD Space app binary.
- MCP development and CI flows now invoke that separate package explicitly when they need the standalone server.
- macOS release verification now checks both the bundle executable and the absence of `gtdspace-mcp` inside `Contents/MacOS`.

## Installation

Download the installer or package that matches your platform:

- Windows: `.msi` installer or `.exe` (NSIS) installer
- macOS Intel: `.dmg` for Intel Macs
- macOS Apple Silicon: `.dmg` for Apple Silicon Macs
- Linux: `.AppImage` (universal) or `.deb` (Debian/Ubuntu)

## Notes

- macOS users should install `v2.0.2`
- The standalone MCP server is still available for local-model workflows through the separate `src-tauri/mcp-server` package
