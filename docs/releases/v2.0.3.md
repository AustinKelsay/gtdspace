# GTD Space {{tag}}

## Highlights

- Fixed the remaining cross-platform MCP test mismatch that caused the Windows CI pipeline to fail after `v2.0.2`.
- Kept the `v2.x` release train aligned by syncing the Rust lockfiles to the shipped app version.

## Notable Changes

- MCP stdio tests now compare workspace roots using the same normalized path format the backend exposes, so Windows verbatim paths no longer cause false failures.
- The Rust lockfiles now correctly reflect the `2.0.2` application version prior to the next patch bump.
- No user-facing GTD Space behavior changed in this patch beyond release and CI reliability.

## Installation

Download the installer or package that matches your platform:

- Windows: `.msi` installer or `.exe` (NSIS) installer
- macOS Intel: `.dmg` for Intel Macs
- macOS Apple Silicon: `.dmg` for Apple Silicon Macs
- Linux: `.AppImage` (universal) or `.deb` (Debian/Ubuntu)

## Notes

- `v2.0.3` is the recommended patch release on top of the `v2.0.x` line.
- This patch is primarily a release-stability and CI-correctness follow-up to `v2.0.2`.
