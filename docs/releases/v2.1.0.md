# GTD Space {{tag}}

## Highlights

- Added a more polished dashboard flow with better selection behavior, bulk actions accessibility, and clearer interactions.
- Hardened MCP server settings, workspace resolution, and validation so partial saved settings and workspace overrides behave more predictably.

## Notable Changes

- Dashboard updates improve action accessibility, selection handling, and overall interaction polish.
- MCP settings now ship with better defaults, clearer validation guidance, and an explicit default for disabling read-only mode unless configured otherwise.
- Workspace and path handling were tightened across settings parsing, config fallback behavior, and MCP workspace validation.
- The MCP schema now strips unsupported `uint` formats so integrations receive a safer schema surface.
- No manual workspace migration is required for this release, but users relying on custom MCP workspace overrides should review their settings after upgrading.

## Installation

Download the installer or package that matches your platform:

- Windows: `.msi` installer or `.exe` (NSIS) installer
- macOS Intel: `.dmg` for Intel Macs
- macOS Apple Silicon: `.dmg` for Apple Silicon Macs
- Linux: `.AppImage` (universal) or `.deb` (Debian/Ubuntu)

## Notes

- `v2.1.0` is the recommended minor upgrade from the `v2.0.x` line.
- This release is focused on dashboard UX improvements and more reliable MCP/workspace configuration handling.
