# GTD Space {{tag}}

## Highlights

- Added a pre-push encrypted backup diff review flow so git sync changes can be inspected before backup pushes are finalized.
- Refined the desktop UI around the dark theme, sidebar reload behavior, and cancelled-project visibility to make day-to-day navigation more reliable.

## Notable Changes

- Git sync now preserves backup timestamps more carefully and handles preview and backup push flows more defensively.
- The app can surface file reloads more clearly, including a dedicated toast path and sturdier sidebar refresh behavior after workspace changes.
- Cancelled projects are now easier to surface from the sidebar and related project views.
- Appearance settings and header controls were simplified as the app styling consolidated around the current dark-mode design direction.
- This release is primarily a workflow and polish upgrade; no manual migration is expected for existing workspaces.

## Installation

Download the installer or package that matches your platform:

- Windows: `.msi` installer or `.exe` (NSIS) installer
- macOS Intel: `.dmg` for Intel Macs
- macOS Apple Silicon: `.dmg` for Apple Silicon Macs
- Linux: `.AppImage` (universal) or `.deb` (Debian/Ubuntu)

## Notes

- `v2.3.0` is the recommended minor upgrade on top of `v2.2.x`.
- Teams using encrypted backup pushes should validate the new diff-review step in their normal git sync flow after upgrading.
