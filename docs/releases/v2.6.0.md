# GTD Space {{tag}}

## Highlights

- Refactored the workspace sidebar controller so project, action, and section state stays in sync more reliably across renames, deletes, and nested project paths.
- Hardened habit history and status syncing so legacy history formats, reset rows, and live editor updates now repair and refresh cleanly instead of drifting out of sync.

## Notable Changes

- Added dedicated sidebar path classification, overlay, loader, and workspace lifecycle hooks to simplify the sidebar architecture and improve reload behavior for project actions.
- Preserved legacy habit tracking notes and prose during history repair, normalized mixed path/event payloads for habit updates, and refreshed dashboard and sidebar habit views more consistently after edits.
- Updated the Tauri dependency set and refreshed the generated desktop and macOS schema outputs to match the shipped plugin surface.

## Installation

Download the installer or package that matches your platform:

- Windows: `.msi` installer or `.exe` (NSIS) installer
- macOS Intel: `.dmg` for Intel Macs
- macOS Apple Silicon: `.dmg` for Apple Silicon Macs
- Linux: `.AppImage` (universal) or `.deb` (Debian/Ubuntu)
